Pass by value:
	passing a copy of a variable with a value. The original variable cannot be modified.
	this requires more memory as a new copy of the variable is created.
	void caller(){
		int input = 10;
		getsCalled(input);
		cout << input << endl; // will still be 10.
	}
	void getsCalled(int input){
	input = 2; // will not change the original variable, only the copy that was sent.
	}

pass by reference:
	the program makes a reference to the variable used as parameter. Doesn't require a copy.
	this requires less memory as no copy is needed.
	
void caller(){
		int input = 10; 
		cout << input << endl; // is currently 10.
		getsCalled(input);
		cout << input << endl; // is currently 2.
	}
	void getsCalled(int *input){
		requires a pointer to the variable that was called. 
		*input = 2; // will change the original variable REFERENCED for the method.
	}

Access modifier:
	fields er private by default.
Class Fish{
	int size;
}
equivalent to
Class Fish{
private:
	int size;
}
